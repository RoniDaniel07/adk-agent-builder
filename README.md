# 🤖 ADK Agent Builder

🚀 Built low-code AI agents using **Google ADK Visual Builder** and deployed with **Cloud Run**

---

## 📌 Overview

This project demonstrates how to create and manage **AI agents using a visual interface** with minimal code.

It follows the official Google codelab:
https://codelabs.developers.google.com/codelabs/create-low-code-agent-with-ADK-visual-builder

---

## 🧠 What This Project Does

* Builds multiple AI agents using ADK
* Uses **Gemini-powered LLM agents**
* Demonstrates **multi-agent workflows**
* Runs locally via ADK Web UI
* Prepares deployment to Cloud Run

---

## 🛠️ Tech Stack

* Python
* Google ADK
* ADK Visual Builder
* Google Cloud Run
